
# Microfrontend Disney

Microfrontend mf2 Disney using react and module federation

## Installation dependencies

Install with yarn

```bash
  cd mf-disney
  yarn
```

## Run Locally

Start the server

```bash
  yarn start
```

Open browser in port <http://localhost:4001>

## Running Tests

To run tests, run the following command

```bash
  yarn test
```

## Generate build

```bash
  yarn build
```
